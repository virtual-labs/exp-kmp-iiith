1 Introduction

This document builds the KMP algorithm demonstration interactive artefact(HTML).

2 Features of the artefact

  • Artefact provides demo for the runtime of KMP String searching algorithm.
  • Artefacts begins with a default text string and pattern string.
  • Artefact allows user to input their own text string and pattern string as well.
  • Artefact contains 2 buttons Next and Setup.
  • Pressing the setup buttons resets the demo with the text present inside the input text fields.
  • Next buttons goes forward with the experiment.

3 HTML Framework of KMP experiment
